História

O HTML foi criado em 1990 por um físico chamado Tim Berners-Lee. Na época, Berners-Lee estava trabalhando no CERN, o laboratório de física de partículas na Suíça.
Ele desenvolveu o HTML como uma maneira de compartilhar documentos com seus colegas de trabalho.

Versões

  1. HTML 2.0 de 1997

HTML 2.0

A primeira versão oficial do HTML foi o HTML 2.0, que foi publicado em 1995.
Esta versão incluía uma série de elementos que ainda são usados hoje, como headings title(títulos), links, listas e formulários.

HTML 3.2

Em 1997, o HTML 3.2 foi lançado. Esta versão introduziu novos elementos, como tabelas e applets, que permitiam a incorporação de conteúdo interativo.

O HTML 4.01, lançado em 1999, trouxe ainda mais melhorias e novos recursos, incluindo suporte para CSS e scripts, bem como melhor acessibilidade e internacionalização.

HTML5

O HTML5 introduziu uma série de novos elementos e APIs que permitiram a criação de aplicações web mais ricas e interativas. Alguns dos novos elementos incluídos no HTML5 são header, footer, article, section, nav, figure, figcaption, video, audio, e muitos outros.
Além disso, o HTML5 também introduziu uma série de APIs JavaScript, como a API de Geolocalização, a API de Armazenamento Web, a API de Histórico, a API de Arrastar e Soltar, e muitas outras.
