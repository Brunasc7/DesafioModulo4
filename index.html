  2. Versões Html
    Priveira Versão
    A primeira versão oficial do HTML foi o HTML 2.0, que foi publicado em 1995.
    Segunda Versão
    Em 1997, o HTML 3.2 foi lançado. Esta versão introduziu novos elementos
    Terceira Versão
    O HTML 4.01, lançado em 1999, trouxe ainda mais melhorias e novos recursos
    Quarta Versão
    No entanto, a maior mudança veio em 2014 com o lançamento do HTML5.
    O HTML5 introduziu uma série de novos elementos e APIs que permitiram a criação de aplicações web mais ricas e interativas.
